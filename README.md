# FloorView

Factory operations dashboard for Muff Manufacturing.

## Features
- Track product stages
- Stage transitions with audit log
- Dashboard statistics
- Bulk updates

## Tech Stack
- Python 3.11
- FastAPI
- PostgreSQL
- asyncpg

## Setup
pip install -r requirements.txt
uvicorn main:app --reload
